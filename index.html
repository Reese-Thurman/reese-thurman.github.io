Capstone Project: The Banking Game

The Banking Game is a text-based financial simulation written in Python that challenges players to manage investments and reach $1,000,000 before retirement. It features compound interest, random financial events, and persistent data storage through SQLite.

  • Software Design & Engineering: Modular architecture, encapsulated functions, and maintainable structure.
  • Algorithms & Data Structures: Compound interest calculation, random event systems, and statistical trend analysis.
  • Databases: SQLite integration for settings, leaderboard persistence, and difficulty customization.
